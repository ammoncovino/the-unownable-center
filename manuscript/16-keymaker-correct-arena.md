# Keymaker: Designing a Correct Arena

Recovered structural placement:

This chapter sketches a non-dominating arena architecture: a platform where contributions are evaluated across multiple dimensions instead of collapsed into a single like, vote, or karma score.

Recovered design direction:

- Clarity: can readers see the claim?
- Evidence: can the claim be checked?
- Good-faith / Non-abusive: is the contribution structurally usable in a shared space?
- Novelty: does it add signal rather than repeat the same performance?

Function:

Keymaker exists to show that systems can be redesigned, not only endured. If Reddit is an arena that trains performance, a correct arena would train clearer claims, stronger evidence, repairable disagreement, and visible uncertainty.

Bridge:

The next chapters return to the Jesus arc and ask what happens when a non-coercive architecture confronts systems that rely on force.
